Set the primary role of this device in the mesh network
Role Information

The device role determines how your device functions in the Meshtastic network:

  • Client: Standard node that sends and receives messages
  • Client (Muted): Receives but doesn't send routine broadcasts
  • Router: Prioritizes routing packets to other nodes
  • Repeater: Specialized router that extends network range
  • Tracker: Primarily sends position updates with GPS priority
  • Sensor: Primarily sends telemetry data
  • TAK Client: Optimized for ATAK system communication
  • TAK Tracker: Combines TAK optimization with tracking features
  • Client (Hidden): Functions as client but doesn't broadcast its presence
  • Lost and Found: Specialized role for lost device recovery
  • Router (Late Join): Router that can join networks late
